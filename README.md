# 🎮 Scouts Games Cup

Formulário de inscrição para o campeonato **Scouts Games Cup**.

## Modalidades
- 🔥 Free Fire (Squad - 4 jogadores)
- ⭐ Brawl Stars (Trio - 3 jogadores)
- ⛏️ Minecraft (Solo/Dupla)
- ⚔️ Wild Rift (Squad - 5 jogadores)

## Stack
- HTML/CSS/JS puro — zero dependências
- Google Sheets como banco de dados via Apps Script
- Hospedado na Vercel

## Deploy
Conecte este repositório à Vercel e o deploy é automático a cada push.
